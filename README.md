## 🤖 About Lesta Bot v2

**Lesta Bot v2** is a powerful, fully-featured WhatsApp automation bot built on the **Baileys** multi-device engine. Originally designed for **Termux on Android**, it works on any Node.js environment including VPS and cloud platforms.

The bot handles **70+ commands** spanning group moderation, YouTube/TikTok downloads, AI chat, sticker creation, games, font styles, TTS, weather, and much more — all triggered by simple WhatsApp messages with the `.` prefix.

> 🔑 **Command Prefix:** `.` — changeable in `config.js`  
> 👑 **Auth Method:** Pairing Code (no QR scan needed)  
> 💡 **No build step required** — runs directly with `node index.js`

## 🔑 Connecting to WhatsApp (Pairing Code)

When you run `npm start` for the first time:

```
╔════════════════════════════╗
║      LESTA BOT v2  🇰🇪      ║
║   WhatsApp Automation      ║
╚════════════════════════════╝

⏳ Requesting pairing code for 254706519089...

╔══════════════════════════════╗
║  PAIRING CODE: AB12CD34    ║
╚══════════════════════════════╝

👉 WhatsApp → Linked Devices → Link with phone number
```

**On your phone:**
1. Open **WhatsApp**
2. Tap ⋮ → **Linked Devices** → **Link a Device**
3. Choose **Link with phone number**
4. Enter your number → enter the **8-digit code**
5. Done! ✅ Bot connects automatically

> ⚠️ Code expires in ~60 seconds. If it expires, press `Ctrl+C` and `npm start` again.

## ⚙️ Configuration Reference

Edit `config.js` before starting:

```js
module.exports = {
  prefix: ".",                     // Command prefix — can be ! / or anything
  botName: "✨ Lesta Bot v2",      // Displayed in menu header
  ownerNumber: "254706519089",     // YOUR number — country code, no + sign
  owners: ["254706519089"],        // Add more owners: ["254...", "1234..."]
  autobioText: "HELLO KENYA 🇰🇪",  // Base text for auto-bio
  autobioInterval: 20000,          // Bio update every 20,000ms = 20 seconds
  followUpMsg: "👋 Send .menu!",   // Auto-sent to first-time DM senders
  badWords: ["..."],               // Words filtered by antibadword
  emojis: ["❤️", "😂", "🔥"],     // Pool for autoreact feature
};
```

## 🔧 Troubleshooting

| ❌ Problem | ✅ Fix |
|---|---|
| Bot logged out | `rm -rf session/ && npm start` |
| Pairing code expired | `Ctrl+C` → `npm start` |
| Downloads not working | `pip install yt-dlp` |
| Video sticker fails | `pkg install ffmpeg` |
| `npm install` errors | `pkg install python` then retry |
| Commands not responding | Check `ownerNumber` in `config.js` |
| Bot crashes instantly | Check Node.js version: `node -v` (needs 18+) |
| Session invalid | Delete `session/` folder and re-pair |
